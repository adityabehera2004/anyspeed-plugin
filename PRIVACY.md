# Privacy Policy

## Data Collection

AnySpeed does not collect, track, or transmit any personal information or usage data.

## Playback Speed Preference

The extension stores your selected playback speed preference (a numeric value between 1.0x and 5.0x) locally on your device using Chrome's secure storage API (`chrome.storage.sync`).

- **Storage**: The speed preference is stored locally on your device
- **Purpose**: The speed value is used solely to control the playback rate of HTML5 video and audio elements on web pages you visit
- **Sharing**: The speed preference is never shared with third parties, sent to any servers, or used for any purpose other than controlling media playback speed
- **Removal**: You can reset your speed preference at any time by adjusting the slider in the extension popup

## No Analytics or Tracking

This extension does not:
- Collect analytics or usage statistics
- Track user behavior or browsing history
- Send data to third-party services
- Use cookies or tracking technologies
- Access any personal information or page content beyond media elements

## Contact

If you have questions about this privacy policy, please contact us through the Chrome Web Store listing.
